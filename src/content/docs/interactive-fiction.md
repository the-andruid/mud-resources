Most of this site focuses on multiplayer text games. If what you really want is to write or play a *single-player* text game, this page is for you.

Solo text games, commonly known as interactive fiction or IF, evolved in parallel to multi-user (MU*) games and thus have a rich history of their own.

This page is meant to serve as a jumping-off point. For deeper coverage, the IF community itself maintains excellent resources (some linked below).

## Finding & playing games

- [IFDB](https://ifdb.org/) - The Interactive Fiction Database; the central catalog, with reviews and recommendations
- [IF Archive](https://ifarchive.org/) - The community's canonical preservation archive: decades of games, tools, interpreters, and source
- [CASA - Classic Adventures Solution Archive](https://solutionarchive.com/) - Walkthroughs, maps, and hints, strong on the commercial 8/16-bit era
- [Key & Compass](https://plover.net/~davidw/) - David Welbourn's large, regularly updated collection of walkthroughs and maps

### Interpreters

Programs that run IF games on your own device.

- [Parchment / iplayif.com](https://iplayif.com/) - Play most Z-machine and Glulx games straight in your browser
- [Gargoyle](https://github.com/garglk/garglk) - Typography-focused desktop player supporting many formats (Z-code, Glulx, TADS, Hugo, Alan, and more)
- [Lectrote](https://github.com/erkyrath/lectrote) - Cross-platform desktop player for Z-code and Glulx
- [Frotz](https://gitlab.com/DavidGriffith/frotz) - The classic, long-maintained Z-machine interpreter

## Authoring tools

### Parser IF

- [Inform 7](https://inform7.com/) - Natural-language design system used for hundreds of award-winning works; [source on GitHub](https://github.com/ganelson/inform)
- [Inform 6](https://inform-fiction.org/) - The earlier, procedural Inform; still maintained and used as a compile target by Inform 7
- [TADS](https://www.tads.org/) - Programmer-oriented system (TADS 2/3); mature and stable
- [Dialog](https://linusakesson.net/dialog/) - Modern Prolog-inspired language compiling to the Z-machine
- [ALAN](https://www.alanif.se/) - Natural-language-flavored IF language, beginner-friendly
- [Adventuron](https://adventuron.io/) - Browser-based, with a retro 8-bit adventure aesthetic
- [Quest & Squiffy](https://textadventures.co.uk/) - Web-based authoring: Quest (parser/hybrid) and Squiffy (choice/markup)
- [ADRIFT](https://www.adrift.co/) - GUI-driven parser-game toolkit requiring no programming
- [Hugo](https://hugoif.github.io/) - A multimedia-capable parser system; resources hub, with the [manual here](https://tajmone.github.io/hugo-book/)
- [ZILF](https://foss.heptapod.net/zilf/zilf) - A modern reimplementation of Infocom's ZIL toolchain, for authoring Infocom-style games
- [PunyInform](https://github.com/johanberntsson/PunyInform) - An Inform 6 library for games that run on vintage 8-bit hardware
- [Gruescript](https://versificator.itch.io/gruescript) - A click-based, "parser-like" web IF language by Robin Johnson

### Choice & hypertext

- [Twine](https://twinery.org/) - The dominant open-source hypertext IF tool; browser-based and beginner-friendly
- [ChoiceScript](https://www.choiceofgames.com/) - Powers the Choice of Games catalog of multiple-choice novels
- [ink](https://www.inklestudios.com/ink/) - Inkle's narrative scripting language, with the Inky editor (and the casual web tool [inklewriter](https://www.inklestudios.com/inklewriter/))
- [Texture](https://texturewriter.com/) - No-code, drag-words-onto-text authoring, well suited to touchscreens
- [Windrift](https://github.com/lizadaly/windrift) - Liza Daly's JavaScript framework for mutable hypertext stories

### Twine story formats

The format you pick determines what a Twine project can do.

- [Harlowe](https://twine2.neocities.org/) - Twine's default format; dynamic hypertext
- [SugarCube](https://www.motoslave.net/sugarcube/2/) - Powerful format for more game-like stories
- [Snowman](https://github.com/videlais/snowman) - Minimal, JavaScript-and-template focused
- [Chapbook](https://klembot.github.io/chapbook/) - Modern and writer-friendly, by Twine's creator

## Competitions & events

- [IFComp](https://ifcomp.org/) - The Interactive Fiction Competition, running annually since 1995
- [Spring Thing](https://www.springthing.net/) - A more relaxed Spring festival, often for longer works
- [IntroComp](https://introcomp.org/) - For introductory excerpts of in-progress games
- [ParserComp](https://parsercomp.itch.io/) - A competition specifically for parser games
- [XYZZY Awards](https://xyzzyawards.org/) - The community "best of the year" IF awards

## Community

- [intfiction.org forum](https://intfiction.org/) - The main IF community forum, covering all systems and languages
- [IFWiki](https://www.ifwiki.org/) - Community wiki: deep history, tool documentation, and culture
- [r/interactivefiction](https://www.reddit.com/r/interactivefiction/) - The IF subreddit
- [Planet-IF](https://planet-if.com/) - Aggregator of IF community blogs
- [Emily Short's Interactive Storytelling](https://emshort.blog/) - The most influential IF craft and criticism blog
- [The Digital Antiquarian](https://www.filfre.net/) - Jimmy Maher's deeply researched history of IF and adventure games
- [Brass Lantern](http://brasslantern.org/) - Long-running adventure-game news and articles
- [SPAG](https://www.spagmag.org/) - The historic IF review magazine (1994-2016), now preserved as an archive
- [PR-IF](http://pr-if.org/) - The Boston-area "People's Republic of Interactive Fiction" group
- [IFTF](https://iftechfoundation.org/) - The nonprofit that stewards IFComp, the IF Archive, IFDB, Twine, the forum, and more
- [NarraScope](https://narrascope.org/) - IFTF's narrative-games conference, with talk recordings posted

## History & preservation

- [The Obsessively Complete Infocom Catalog](https://eblong.com/infocom/) - Andrew Plotkin's effort to collect every version of every Infocom game
- [historicalsource](https://github.com/historicalsource) - Jason Scott's archive of original Infocom ZIL source code
- [Get Lamp](http://www.getlamp.com/) - Jason Scott's documentary about text adventures; the film and roughly 80 raw interviews are [on the Internet Archive](https://archive.org/details/GET_LAMP_The_Text_Adventure_Documentary)
- [From Infocom to 80 Days: An Oral History of Text Games and Interactive Fiction](https://arstechnica.com/gaming/2024/06/from-infocom-to-80-days-an-oral-history-of-text-games-and-interactive-fiction/) - Ars Technica's 2024 feature: fifty years of IF told by its creators, from Infocom to Inkle
- [Magnetic Scrolls Memorial](https://msmemorial.if-legends.org/) - History and preservation for the British IF house Magnetic Scrolls
- [Level 9 Memorial](https://l9memorial.if-legends.org/) - Tribute to the British IF company Level 9
- [Strand Games](https://strandgames.com/) - Modern IF studio publishing the official Magnetic Scrolls remasters
- [Scott Adams Grand Adventures](http://www.msadams.com/) - Official site of text-adventure pioneer Scott Adams

## Academic & electronic literature

- [Nick Montfort](https://nickm.com/) - IF scholar and poet; home of *Twisty Little Passages*, the first book-length study of IF
- [Electronic Literature Organization](https://eliterature.org/) - Research and advocacy for born-digital literary work
- [Electronic Literature Collection](https://collection.eliterature.org/) - ELO's multi-volume anthology
- [Eastgate / Storyspace](http://www.eastgate.com/) - Publisher of classic hypertext fiction and the Storyspace authoring tool

## Learning

- [Writing with Inform](https://ganelson.github.io/inform-website/) - The official Inform 7 manuals
- [The Inform Designer's Manual (DM4)](https://inform-fiction.org/manual/about_dm4.html) - The canonical Inform 6 reference
- [The Twine Cookbook](https://twinery.org/cookbook/) - Official, community-maintained Twine how-to recipes

## Tooling & utilities

- [Trizbort.io](https://trizbort.io/) - Browser-based IF map-maker with code export to many systems
- [Borogove](https://borogove.app/) - A browser IDE supporting many IF systems (Inform, Hugo, Dialog, ink, ZIL, and more)
- [Vorple](https://vorple-if.com/) - An HTML/JavaScript multimedia layer for Inform and other parser IF

## Technical specifications

References for interpreter and tool developers.

- [The Z-Machine Standards Document](https://inform-fiction.org/zmachine/standards/) - The spec for Infocom's Z-machine, the most widely used IF virtual machine
- [Glulx](https://eblong.com/zarf/glulx/) - A modern 32-bit IF virtual machine
- [Glk](https://eblong.com/zarf/glk/) - The IF input/output interface standard
- [Blorb](https://eblong.com/zarf/blorb/) - The standard for bundling a game with its graphics and sound
- [Treaty of Babel](https://github.com/iftechfoundation/babel-tool) - The cross-format bibliographic standard that defines the IFID

## Non-English communities

- [CAAD](https://caad.club/) - The principal Spanish-language IF hub ("aventuras conversacionales"), with an active magazine and forum
- [Fiction-interactive.fr](https://www.fiction-interactive.fr/) - The francophone IF community and home of its annual competition
- [OldGamesItalia - IF Italia](https://www.oldgamesitalia.net/) - The main Italian IF hub; the wider site is in hibernation but the IF forum stays active
- [ifwizz.de](https://ifwizz.de/) - An active German IF database and news site
- [INSTEAD](https://instead3.syscall.ru/) - Peter Kosyh's open-source Russian "textographic" IF engine (the site may show a certificate warning)
