Most of this site focuses on multiplayer text games. If what you really want is to write or play a *single-player* text game, this page is for you.

Solo text games, commonly known as interactive fiction or IF, evolved in parallel to multi-user (MU*) games and thus have a rich history of their own.

This page is meant to serve as a jumping-off point. For deeper coverage, the IF community itself maintains excellent resources (some linked below).

## Authoring tools

Tools for writing your own interactive fiction (IF), grouped roughly by approach.

### Hypertext / branching narrative

- [Twine](https://twinery.org/) - Browser-based, beginner-friendly, and free
- [Inklewriter](https://www.inklestudios.com/inklewriter/) - Free tool for writing and publishing interactive stories
- [ChoiceScript](https://www.choiceofgames.com/make-your-own-games/choicescript-intro/) - Powers the Choice of Games catalog

### Parser IF

The classic "you are in a dark room, type EXAMINE LAMP" style.

- [Inform 7](https://ganelson.github.io/inform-website/) - Natural-language authoring system used for hundreds of award-winning works
- [TADS 3](https://www.tads.org/) - Programmer-oriented parser IF system
- [Dialog](https://linusakesson.net/dialog/) - Modern Prolog-inspired parser IF language
- [Adventuron](https://adventuron.io/) - Browser-based, retro 8-bit adventure aesthetic

### Other notable systems

- [Quest](http://textadventures.co.uk/quest/) - Browser-based authoring with both GUI and code modes
- [Squiffy](http://textadventures.co.uk/squiffy/) - Lightweight branching narrative tool from the Quest team

## Where to publish and find games

- [IFDB](https://ifdb.org/) - The Interactive Fiction Database
- [IF Archive](https://www.ifarchive.org/) - The community's preservation archive; decades of games, tools, and source code
- [itch.io interactive fiction tag](https://itch.io/games/tag-interactive-fiction) - Where many indie IF authors publish today
- [Choice of Games](https://www.choiceofgames.com/) - Publisher of ChoiceScript-based games, with a "Hosted Games" label for community-submitted work

## Competitions

- [IFComp](https://ifcomp.org/) - The Interactive Fiction Competition, running since 1995
- [Spring Thing](https://www.springthing.net/) - Spring counterpart to IFComp, more relaxed

## Community

- [intfiction.org forum](https://intfiction.org/) - The main IF community forum
- [IFWiki](https://www.ifwiki.org/) - Community wiki, deep history and tool documentation
- [r/interactivefiction](https://www.reddit.com/r/interactivefiction/) - Subreddit
- [Planet IF](https://planet-if.com/) - Aggregator of IF blogs
- [50 Years of Text Games](https://if50.substack.com/) - Stroll through the history of text games, from interactive fiction to MUDs
